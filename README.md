## Installation

Run the following commands to harden your Amazon Linux 2 instance.

```bash
git clone https://github.com/massyn/amazon-linux-2-cis-benchmark/
cd amazon-linux-2-cis-benchmark/
sudo ./amzn2-cis.sh
```
